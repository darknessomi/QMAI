# 角色 Skill 索引

## 标准模板

所有人物角色文件必须优先遵循 `CHARACTER_PROFILE_STANDARD.md`。角色文件不是普通人物小传，而是 AI 写章纲和正文时读取的角色状态卡、语言风格卡和出场追踪卡。

- `character-design`：角色定位、欲望、恐惧、信念、缺陷。
- `relationship-emotion`：感情线、关系拉扯、好感度门槛、情绪价值。
- `supporting-cast`：配角、反派、导师、盟友、工具角色反工具化。

## 核心原则

- 角色必须有故事功能和情绪功能。
- 重要角色必须记录当前状态、语言风格、关系变化和出场记录。
- 男主、女主、男配、女配、反派、导师、上级、盟友、情感型配角可以分文件夹管理。
