# 长篇正文写作

## 适用范围

长篇正文、连载章节、按章纲写正文、承接上一章、推进主线。

## 写作规则

1. 先读取已确认章纲，严格遵循本章目标、核心事件、场景顺序和章尾钩子。
2. 开篇承接上一章人物位置、情绪、未解决事件和钩子。
3. 每个场景都要推进事件、人物或伏笔，不能只写气氛。
4. 爽点、虐点、信息点要有节奏，不要一章平铺。
5. 结尾必须制造下一章期待。

## 禁止

- 改写已确认章纲主事件。
- 临时新增万能设定解决冲突。
- 输出分析、说明、写作过程或 Skill 名称。
