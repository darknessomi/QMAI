# 战斗与动作

## 适用范围

玄幻战斗、都市高武、动作戏、追逐、对决、比赛、打脸场景。

## 战斗流程

1. 目标：双方为什么打。
2. 差距：力量、信息、环境、资源差异。
3. 试探：第一轮行动暴露能力边界。
4. 变化：误判、底牌、环境利用或第三方介入。
5. 代价：伤势、消耗、身份暴露、资源损失。
6. 结果：胜负如何改变关系和主线。

## 写作要求

- 不只写招式名，要写动作、反应和结果。
- 爽点来自策略和反转，不只是数值更高。
- 围观反应要分层，不要全部喊“怎么可能”。
