# 对话与情绪

## 适用范围

对话、潜台词、情绪外化、关系拉扯、角色声线。

## 对话规则

1. 每句对话至少承担一个功能：试探、拒绝、隐瞒、攻击、安抚、推进信息。
2. 不让角色替作者解释设定。
3. 不同角色说话方式要有身份、关系、情绪差异。
4. 潜台词优先，能用动作和停顿表达的不要全说出来。

## 情绪外化

- 情绪词后必须接具体动作、物件或场景反应。
- 激烈情绪写短句和动作，沉淀情绪写具体细节。
- 不用“他很难过”结束段落，要写难过如何改变行动。
