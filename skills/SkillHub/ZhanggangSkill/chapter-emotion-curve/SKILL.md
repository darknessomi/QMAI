# 情绪曲线

## 核心原则

一章不能从头到尾停在同一种情绪里。使用顺滑曲线：开局状态、中段变化、峰值爆发、结尾沉淀并保留期待。

## 四个节点

1. **开局情绪**
   - 承接上一章情绪，不要突然切断。
   - 匹配章节属性和入场条件。

2. **中段变化**
   - 让冲突推动情绪上升、下降、收紧或变得更不确定。
   - 用触发条件或互动条件制造情绪转折。

3. **峰值爆发**
   - 最强情绪释放放在核心爆点，不要随意提前。
   - 峰值必须对齐本章核心记忆点。

4. **结尾沉淀**
   - 让情绪有一点呼吸空间，但保留期待线。
   - 用结尾钩子保留一个未解决问题。

## 常用曲线

- 平开 -> 冲突上升 -> 高潮释放 -> 伏笔降落。
- 冲突开篇 -> 小解释缓冲 -> 再次上升 -> 反转收尾。
- 憋屈开篇 -> 压迫升级 -> 打脸释放 -> 新问题钩子。

```
|---|---|---|---|

```

## 常见问题

- 整章都很平，没有起伏。
- 整章都很炸，让读者疲劳。
- 情绪峰值早于冲突成熟。
- 结尾完全归零，没有期待线。
