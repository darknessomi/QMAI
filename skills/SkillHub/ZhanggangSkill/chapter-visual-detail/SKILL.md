# 画面细节

## 核心原则

读者需要看到画面，而不只是读到解释。画面感来自三类细节：环境细节、动作微细节、信息暗线细节。

## 三类细节

| 类型 | 作用 | 用法示例 |
|---|---|---|
| 环境细节 | 托住情绪和处境 | 暴雨压抑、烛火摇晃、长街空冷 |
| 动作微细节 | 用身体表现心理 | 攥拳、指尖发白、肩膀轻颤、避开视线 |
| 信息暗线细节 | 隐藏线索和伏笔 | 异常物件、破绽、旁人一句无心台词 |

## 执行步骤

1. **用可观察行为替代抽象心理**
   - 弱：“他很生气。”
   - 强：“他低着头，指节一点点收紧，指尖泛白。”

2. **让环境绑定情绪**
   - 不写装饰性天气；环境要形成压力、反差或情绪揭示。
   - 优先使用章节关键词里已经暗示的环境。

3. **把有用信息藏进细节**
   - 让读者回头看时觉得“原来如此”。
   - 使用关键词条件里的必须插入线索和道具。

4. **每章至少放一个动作微细节**
   - 优先放在情绪转折或核心爆点附近。
   - 最强微细节放在峰值或反转处。

## 输出格式

```
## 画面细节

### 环境细节

### 动作微细节

### 信息暗线细节

### 可替换的抽象心理描写
```

## 常见问题

- 解释情绪，而不是展示情绪。
- 加入无关风景，不能影响气氛。
- 场景里没有任何隐藏信息。
- 核心爆点只靠内心独白完成。
