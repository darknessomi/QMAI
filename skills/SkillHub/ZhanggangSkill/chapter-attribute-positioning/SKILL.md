# 章节定位

## 核心原则

写章纲前，先确定读者应该感受到什么，以及这一章带来什么变化。章节属性 = 情绪标签 + 功能定位。

## 执行步骤

1. **选择一个核心情绪标签**
   - 可选方向：逆袭、打脸、悬念、憋屈、反转、糖点、燃点、紧张、共情。
   - 只选一个主标签，辅助情绪必须服务主标签。

2. **选择一个章节属性**
   - 可选方向：铺垫章、冲突章、反转章、低谷章、蓄力章、高潮章、过渡章。
   - 属性决定节奏：高潮章不能平，过渡章也必须有实际变化。

3. **定义四个功能**
   - 主线承接：承接上一章的剧情状态，不能断档。
   - 主线推进：本章完成了什么进度、破局、时刻或线索。
   - 人物变化：主角心态、能力、认知、短板或关系发生什么变化。
   - 追读目的：解决上一章什么疑问，又埋下下一章什么期待。

4. **用五句话以内写清定位**
   - 写不短，通常说明这一章不够聚焦。

## 输出格式

```
## 章节定位

### 核心情绪标签

### 章节属性

### 主线承接

### 主线推进

### 人物变化

### 追读目的
```

## 常见问题

- 一章想承载所有情绪，导致重点散。
- 把“过渡章”写成没有任何变化。
- 高潮章只有解释，没有冲突或爆点。
- 忘记结尾要制造读者期待。
