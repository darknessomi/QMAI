# 设定 Skill 索引

## 标准模板

所有设定文件必须优先遵循 `SETTING_PROFILE_STANDARD.md`。设定文件不是背景资料堆砌，而是 AI 写大纲、章纲、人物和正文时读取的规则约束库。

- `idea-market-positioning`：灵感整理、题材定位、核心卖点、市场承诺。
- `world-rules`：世界观规则、禁忌、制度、信息公开度。
- `power-system`：力量体系、金手指、能力边界、代价。
- `faction-system`：势力、组织、门派、家族、公司、阵营。
- `foreshadowing-suspense`：伏笔、悬念、线索、回收计划。
- `map-progression`：地图推进、换地图、环境升级。

## 核心原则

- 题材定位决定读者承诺，后续大纲、章纲、正文不能偏离。
- 世界规则、力量体系、金手指必须写清边界、代价和反制点。
- 势力、地图、伏笔必须有更新记录，便于连续章节读取最新状态。
