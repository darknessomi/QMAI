# 力量体系与金手指

## 适用范围

修炼体系、异能、系统、金手指、等级、功法、道具、职业能力。

输出力量体系或金手指文件时必须遵循上级目录的 `SETTING_PROFILE_STANDARD.md`，重点写清获取方式、任务/成长机制、限制、代价、反制点和防崩坏写作约束。

## 设计清单

1. 核心能力：一句话说明主角最有趣的能力用法。
2. 获取方式：天赋、系统、传承、训练、资源、契约或代价。
3. 限制：触发条件、消耗、冷却、风险、暴露、反制。
4. 成长路径：能力如何升级，升级后解决什么新问题。
5. 爽点循环：压力出现、能力触发、选择、兑现、引发更大压力。
6. 使用史：每次解锁、任务、奖励或突破都要能追踪。

## 章纲要求

- 本章能力使用必须产生具体结果。
- 不能只写“突破”，要写突破的原因、代价和后续影响。
- 敌人必须有可见反制或误判。

## 文件要求

- 力量体系默认输出到 `设定/世界观/力量体系.md`。
- 金手指默认输出到 `设定/世界观/金手指.md`。
- 金手指不能直接替主角跳过关键过程。
