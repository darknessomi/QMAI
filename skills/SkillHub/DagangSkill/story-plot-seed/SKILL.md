# Webnovel Plot Seed

## Core Principle

Turn a vague idea into a compact plot skeleton before any detailed settings. The plot seed must answer: who the protagonist is, where they start, what changes, what they pursue, what blocks them, and what final direction the story promises.

## Inputs

Ask only for missing essentials:

- 类型：玄幻、都市、言情、悬疑、科幻、历史等。
- 主角：身份、处境、缺口。
- 触发事件：意外获得什么、失去什么、被迫面对什么。
- 终局方向：登顶、复仇、破案、创业、守护、逃离等。

## Execution

1. **Extract the story engine**
   - Identify protagonist, starting weakness, opportunity, conflict, and long-term direction.

2. **Write the 500-word plot skeleton**
   - Keep it broad enough to expand.
   - Include the main contradiction and the promise of escalation.

3. **Check completeness**
   - If no protagonist action exists, revise.
   - If it reads like only a setting, revise.

## Output Format

```markdown
## 剧情骨架

### 一句话梗概

### 500 字剧情骨架

### 主线驱动力

### 当前缺口
```

## Common Mistakes

- Writing world history instead of plot.
- Giving only mood and aesthetics.
- Creating too many factions before the protagonist has a goal.
