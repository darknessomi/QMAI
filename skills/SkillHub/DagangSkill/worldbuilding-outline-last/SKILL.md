# Webnovel Worldbuilding Last

## Core Principle

Worldbuilding comes last. Add only settings that support the plot, selling point, goal ladder, and character functions. Do not write an encyclopedia before the story can move.

## Execution

1. **Read prior components**
   - Plot seed, selling point, goals, protagonist, and supporting cast must exist.

2. **Derive necessary settings**
   - What rules make the selling point work?
   - What institutions create the stage goals?
   - What locations enable the first arcs?
   - What limits prevent the protagonist from solving everything too easily?

3. **Write minimum viable worldbuilding**
   - Keep only rules that will appear in early story or drive conflict.

4. **Cut decorative lore**
   - If a setting does not affect choices, conflict, or reader expectation, mark it as later.

## Output Format

```markdown
## 世界观设定

### 必要规则

### 势力和地点

### 等级或资源体系

### 与卖点的连接

### 暂不展开的设定
```
